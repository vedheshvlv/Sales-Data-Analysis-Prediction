# Sales Data Analysis and Prediction

## Project Overview
This project analyzes retail sales data and predicts future sales using machine learning techniques.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Visualization using Matplotlib and Seaborn
- Prediction using Linear Regression and Random Forest
- Performance Evaluation (MAE, RMSE)

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## How to Run

1. Install dependencies:
pip install pandas numpy matplotlib seaborn scikit-learn

2. Run the program:
python analysis.py

## Output
- Sales distribution graph
- Correlation heatmap
- Prediction vs Actual graph
- MAE and RMSE values
